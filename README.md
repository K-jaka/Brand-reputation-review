# 2023 Brand Reputation Monitor

An AI-powered dashboard built with **Streamlit** library that analyzes customer sentiment and brand reputation using **Hugging Face Transformers**.

## Features
- **Sentiment Analysis:** Categorizes reviews into Positive/Negative using DistilBERT.
- **Interactive Visualizations:** Monthly filtering and sentiment distribution charts.
- **Word Clouds:** Visualizes key conversation topics.
- **Data Cleaning:** Automated processing of product names and dates.

## Setup Instructions
1. Clone the repository: `git clone https://github.com/K-jaka/Brand-reputation-review`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
